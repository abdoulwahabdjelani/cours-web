  FRONT-END

 Front-end désigne les éléments d'un site ou d'une application que les utilisateurs voient à l'écran et avec lesquels ils vont interagir.

À titre d'exemple, tout ce que les internautes vont voir sur un site internet.

Il se doit de maîtriser au moins trois langages de programmation que sont:

  1.  HTML
  2.  CSS
  3.  JavaScript
