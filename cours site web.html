LES DIFFÉRENTS LANGAGES UTILISÉ EN DÉVELOPPEMENT WEB FRONT-END.


HTML:« HyperText Markup Language ».C’est un langage essentiel. Il indique au site Web quel contenu afficher et, dans une certaine mesure, comment l’afficher.

CSS::« Cascading Style Sheets » est un langage de «style». Il va notamment permettre le changement de couleurs, de polices, le positionnement des éléments ainsi que le changement de taille du texte.

JavaScript:va rendre votre site Web réactif, interactif et attrayant pour vos visiteurs.
