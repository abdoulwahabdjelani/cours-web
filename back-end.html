LE BACK-END


  c'est quoi le back-end ?


le développeur back-end est en charge de l’ensemble des fonctionnalités d’un site web. A la différence du Développeur Front-End, le Développeur Back-End ne s’occupe pas de l’aspect ergonomique du site mais de l’aspect technique et fonctionnel.

Le développeur back-end travaille dans l’ombre il peut concevoir un site web et dans ce cas réaliser lui-même l’architecture technique du site ; ou bien rester sur la plateforme instaurée et adapter des solutions techniques sur celle-ci.

Ils doivent créer le code et les programmes qui alimentent le serveur du site web, les bases de données et toutes les applications que contient le site internet. Ils utilisent un large éventail de langages différents côté serveur afin de construire des programmes compliqués.
